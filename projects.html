Платформа для анализа здоровья

Платформа для анализа текущего здоровья. Пользователь по результатам анализа может посмотреть текущее состояние здоровья. Может спрогнозировать изменения параметров крови и состояния своего здоровья. Также данная платформа позволяет пользователю узнать свой биологический возраст, составить свое питание. Система предназначена для частных клиник, но и любой желающий пользователь может воспользоваться данным сервисом
